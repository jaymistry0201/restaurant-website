# Responsive Food and Restaurant Website (Bootstrap 5)

This is a fully responsive restaurant website built using HTML, CSS, JavaScript, and Bootstrap 5.  
It is designed for restaurants, cafes, and food-related businesses to showcase their menu, services, and team in a simple and visually appealing layout.

## Features

- Responsive layout for all screen sizes  
- Smooth scrolling navigation bar  
- Animated counter section for displaying key statistics  
- Food menu display with image cards  
- Testimonial slider using Bootstrap carousel  
- Team section to introduce staff or chefs  
- Table reservation call-to-action section  
- Clean and organized UI design

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5  
- Font Awesome
